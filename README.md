This repo contains the MCMC scheme for the apper Graphical Model-based clustering. 

- The main algorithm is in the file "gibbs_new.R"; the other files contain auxiliary functions;
- The file "votes.data" contains the dataset analyzed in Section 6.2; the cystic fibrosis dataset analyzed in Section 6.1 is available in the R package "gap.dataset" named as "cf".
